# 🖼️ Image Classification (이미지 분류) 학습 노트

> NVIDIA DLI - Getting Started with Deep Learning

---

## 📌 핵심 개념 요약

- 이미지 분류란 입력 이미지가 어떤 클래스에 속하는지 예측하는 태스크
- CNN(Convolutional Neural Network)을 주로 사용
- 출력층에 **Softmax** 활성화 함수를 사용하여 확률값으로 변환

---

## 🔧 실습 내용

| 항목 | 내용 |
|---|---|
| 사용 프레임워크 | Keras / TensorFlow |
| 주요 레이어 | Conv2D, MaxPooling2D, Flatten, Dense |
| 출력 활성화 함수 | Softmax |
| 손실 함수 | Categorical Cross-Entropy |
| 평가 지표 | Accuracy |
